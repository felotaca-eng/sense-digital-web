FAQ

Preguntas frecuentes

¿Trabajan desde cero o sobre una web existente? +
Ambas. Podemos rediseñar y optimizar una web actual o construir una nueva desde la estrategia y arquitectura inicial.
¿Incluyen contenido y copywriting? +
Sí. Podemos apoyarte con estructura, copy comercial, llamados a la acción y lineamientos visuales para que la web venda mejor.
¿La web queda lista para campañas? +
Sí. Dejamos implementados formularios, eventos, píxeles y medición para que marketing pueda operar desde el día uno.
¿Puedo editar el contenido después? +
Según el alcance, podemos entregarte un sitio editable o acompañarte con un esquema de soporte mensual.
